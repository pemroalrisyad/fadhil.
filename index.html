Tentang Saya

Halo! Nama saya Fadhil Ataillah Merenda.

Saya seorang yang memiliki hobi bermain game. Game bukan hanya sarana hiburan, tetapi juga media yang mengajarkan saya banyak hal, seperti strategi, keterampilan dalam pengambilan keputusan, serta kerja sama tim yang solid.

Saat ini, saya bersekolah di SMP Al Irsyad Purwokerto. Di sini saya belajar banyak hal, baik dalam aspek akademik maupun dalam pembentukan karakter. Lingkungan sekolah yang mendukung membuat saya merasa semakin berkembang.

Hobi Saya

Salah satu hobi utama saya adalah bermain game. Saya suka game strategi, petualangan, dan game online multiplayer. Bermain game membuat saya belajar kerja tim, berpikir cepat, dan bersenang-senang di waktu luang.

Sekolah Saya

Saya saat ini bersekolah di SMP Al Irsyad Purwokerto. Sekolah ini memiliki lingkungan yang islami, fasilitas yang lengkap, dan guru-guru yang ramah serta berpengalaman. Di sini saya belajar banyak hal baik dalam akademik maupun akhlak.
